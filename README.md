# When Does Train-Time Masking Help?

Official code and compact reproducibility bundle for the paper (TBD).

## Overview

This public snapshot is a curated release of the research repository. It keeps the code, configs, manuscript-facing summaries, and rendered figures needed to inspect and reproduce the paper, while excluding heavyweight raw artifacts and internal working files that do not belong in the official public codebase.

## Included

- `src/lab/`
  - core implementation for baselines, MAIT, evaluation, reporting, and pipeline orchestration
- `configs/`
  - dataset, protocol, method, robustness, reporting, ablation, and study configs
- `scripts/`
  - official experiment, aggregation, audit, and figure-generation entrypoints
- `results/lineage_*/{aggregated,audits,mar_overlay,mask_sweep,lambda_sweep}`
  - compact per-study summaries that back the paper’s reported numbers
- `paper/submission_summary/`
  - cross-study manuscript-facing JSON and Markdown summaries
- `figures/` and `paper/figures/`
  - rendered figures used by the manuscript and release bundle
- `access.tex` and `references.bib`
  - manuscript source and bibliography
- `tests/`
  - consistency and regression checks for the released code path

## Intentionally Excluded

- `results/*/raw/`
  - large per-seed run artifacts
- `results/*/logs/`, `results/*/manifests/`, `results/*/tables/`, `results/*/figures/`
  - intermediate or convenience outputs that can be regenerated
- internal notes, review comments, and one-off recovery scripts

This keeps the public release compact while preserving the information needed to verify the paper’s claims.

## Environment Setup

Run commands from the repository root:

```bash
python -m pip install -r requirements-research.txt
export PYTHONPATH=src
```

The experiments fetch benchmark data from OpenML at runtime.

## Recommended Checks

Run the paper-consistency test suite first:

```bash
PYTHONPATH=src python -m unittest tests.test_draft_code_consistency
```

## End-to-End Reproduction

To rerun the full five-dataset study suite and regenerate the manuscript-facing derived outputs:

```bash
python scripts/run_all_studies.py --include-extras
```

That command covers:

- nominal baselines
- the MAIT method and ablations
- robustness evaluation
- calibration evaluation
- significance testing
- mask-rate sweeps
- MAR probes
- Adult and Covertype reconstruction-weight sweeps
- cross-study summary aggregation
- canonical degradation-curve figure generation

The canonical manuscript degradation figures are generated by:

```bash
python scripts/make_composite_figures.py
```
